# Misinformation Detection & Feed Ranking

This project built machine learning classifiers to detect low-reliability social media content and tested how those predictions could be incorporated into a mock X-style feed ranking system.

## Tools
- Python
- Pandas
- NumPy
- scikit-learn
- TensorFlow/Keras
- KerasTuner

## Models
- KNN
- SVM
- Logistic Regression
- CNN-LSTM
- Hybrid ML/DL models

## Evaluation
